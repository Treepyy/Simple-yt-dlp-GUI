# Simple-yt-dlp-GUI
A custom python GUI for [yt-dlp](https://github.com/yt-dlp/yt-dlp) with 4 basic modes: standard video, spliced video, mp3, and custom command.

How to use:
1. Clone the repo
2. Put the [yt-dlp executable](https://github.com/yt-dlp/yt-dlp/releases/tag/2026.02.04) and [ffmpeg binary](https://www.ffmpeg.org/download.html) within the repository.
3. Run the GUI using ```python ui.py```
